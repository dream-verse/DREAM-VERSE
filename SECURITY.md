# Security Policy

## Supported Versions

The latest version of DREAMVERSE is actively maintained and supported.

| Version | Supported |
|---------|-----------|
| Latest  | ✅ |
| Older Versions | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability in DREAMVERSE, please report it responsibly.

📧 Email:
dreamverse.web@gmail.com

Please include:

- Description of the vulnerability
- Steps to reproduce it
- Screenshots (if applicable)
- Browser and device information
- Any suggested fix (optional)

We will review all valid reports as quickly as possible.

Please do not publicly disclose security issues until they have been investigated and resolved.

Thank you for helping keep DREAMVERSE secure.
